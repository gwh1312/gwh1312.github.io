# vuepress搭建博客

# 一、介绍

vuepress是vue全家桶成员之一，尤雨溪大神于2018年4月12日推出。 它是一个vue驱动的静态网站生成器, 它的有点就是以Markdown为中心，几乎零配置，高性能 github地址：https://github.com/yyx990803 vuepress中文网：https://www.vuepress.cn/ 这个中文网就是用vuepress生成的

# 二、环境配置

1. node
2. git

# 三、开始搭建

  1. 本地安装Vuepress
npm install -D vuepress
  1. 创建并进入项目
mkdir vuepress-demo && cd vuepress-demo
  1. 初始化项目
npm init -y

在生成的package.json中，添加如下两个启动命令：

"script":{
    "dev":"vuepress dev docs",
    "build":"vuepress build docs"
}

  1. 创建基本的项目结构 这里推荐看官方文档中推荐的目录机构

  2. 配置config.js

    module.exports = {
    title:"白日梦想家",
    head:[
        ["link",{rel:"icon",href:"/hali.jpg"}],
    ],
    themeConfig:{
        logo:'/tom.jpg',
        nav:[
            {text:'首页',link:'/'},
            {text:'技术文档',
			items:[
			{text:'HTML',link:'/tech/HTML/'},
			{text:'CSS',link:'/tech/Css/'},
			{text:'Javascript',link:'/tech/interview/'},
			{text:'Vue',link:'/tech/Vue/'},
            {text:'React',link:'/tech/React/'}]},
            {text:'博客搭建',link:'/blog/vuepress/'},
            {text:'环境配置',link:'/env/env/'},
            {text:'博客园',link:'https://www.cnblogs.com/gwh1312/'}
        ],
        sidebar:'auto', // 侧边栏配置
    }
}
    1. 启动项目
    npm run dev

    默认服务器启动在了http://localhost:8080/

    1. 配置首页：
    ---
home: true
heroImage: /spider.png
heroText: 我的主页
tagline: My homepage
actionText: 技术文档 →
actionLink: /tech/interview/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
footer: xxx的个人博客
---
    1. 变更主题

    对默认主题不满意，想换个别的主题？安排！ 在config.js中设置：

    module.exports = {
  theme: 'vuepress-theme-xx'
}

    可以在npm中寻找自己喜欢的主题 ，我们如果觉得这些主题不适合你，你也可以自己开发

# 四、部署上线

  1. Github Pages

在这里就使用 Github Pages,即 Github 提供的、用于搭建个人网站的静态站点托管服务。很多人用它搭建个人博客。这种方式的好处是免费、方便，坏处是速度可能会有些慢、不能被国内的搜索引擎收录。

  1. github创建仓库

    ​ 1. 登录 github

    ​ 2. 新建仓库一：username.github.io

     （必须为你的github账户的username，而不是昵称啥的）

    ​ 3. 新建仓库二，名称随意如vuepress-demo

    二者的关系是：仓库一负责显示网站内容，我们不需要改动它；日常开发和新增内容，都在仓库二中，并通过 npm run deploy 命令，将代码发布到仓库一

  2. 关联本地项目与github仓库

// 先cd到你的demo
cd vuepress-demo
// git初始化
git init
// 关联github仓库
git remote add origin git@github.com:xxx/vuepress-demo.git
  1. 新建部署文件

在跟目录下新建deploy.sh:

#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run build

# 进入生成的文件夹
cd docs/.vuepress/dist

# 如果是发布到自定义域名
# echo 'www.yourwebsite.com' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果你想要部署到 https://USERNAME.github.io
git push -f git@github.com:xxx/xxxx.github.io.git master

# 如果发布到 https://USERNAME.github.io/<REPO>  REPO=github上的项目
# git push -f git@github.com:USERNAME/<REPO>.git master:gh-pages

cd -
  1. git提交
  2. 新建deploy指令并执行

package.json文件夹中添加发布命令

"scripts": {
  "deploy": "bash deploy.sh"
}
  1. 发布成功

查看自己的博客域名：https://xxx.github.io/
